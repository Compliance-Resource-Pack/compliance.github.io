Definisi

Paket dapat dianggap sebagai paket tekstur untuk streamer atau paket tekstur youtuber jika dibuat oleh atau atas nama streamer atau youtuber.

Distribusi Paket

Jika anda ingin mendistribusikan paket berikut (salinan modifikasi maupun salinan identik), anda harus mematuhi syarat dan ketentuan berikut:

  1. Anda harus memiliki izin dari Staf Tim Compliance.
  2. Anda harus mentautkan kembali ke https://www.compliancepack.net di tempat yang pantas.
  3. Anda tidak boleh membuat uang dari paket tekstur ini dengan segala bentuk.
  4. Anda harus menyertakan lisensi ini di semua salinan.
